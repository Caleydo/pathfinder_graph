=====================

Pathfinder python server plugin for communicating with [Neo4j](https://neo4j.com/).

Installation
------------

```
git clone https://github.com/phovea/pathfinder_graph.git
cd pathfinder_graph
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
